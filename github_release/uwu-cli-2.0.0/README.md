## üöÄ Installation

### Via pip (from GitHub)
```bash
pip install git+https://github.com/UwU-CLI/UwU-Cli.git
```

### Via pipx (recommended)
```bash
pipx install git+https://github.com/UwU-CLI/UwU-Cli.git
```

### Manual Installation
```bash
git clone https://github.com/UwU-CLI/UwU-Cli.git
cd UwU-Cli
pip install -e .
```
